## **Balises :**
*	`<html>`
*	`<head>`
*	`<title>`
*	`<body>`
*	Rubriques (`h1`, `h2`, `h3`, `h4`, `h5`, `h6`)
*	Paragraphe `<p>`
*	Gras `<strong>`, italique `<em>`
*	Liste ordonnée `<ol>`, Liste non-ordonnée `<ul>`, Élément de liste`<li>`
*	Saut de ligne`<br>`, Ligne horizontale `<hr>`
*	Image `<img>` et attributs : `alt`, `src`, `width`, `height`
*	Ancre `<a href="">`
